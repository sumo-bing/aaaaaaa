编程知识扩展：C 语言是一种通用的、面向过程式的计算机程序设计语言。   是一种广泛使用的计算机语言，它与 Java 编程语言一样普及，二者在现代软件程序员之间都得到广泛使用。
      1.  所有的 C 语言程序都需要包含 main() 函数。 代码从 main() 函数开始执行。
      2.  /* ... */ 用于注释说明。
      3.  printf() 用于格式化输出到屏幕。printf() 函数在 "stdio.h" 头文件中声明。
      4.  stdio.h 是一个头文件 (标准输入输出头文件) , #include 是一个预处理命令，用来引入头文件。 当编译器遇到 printf() 函数时，如果没有找到 stdio.h 头文件，会发生编译错误。
      5.  return 0; 语句用于表示退出程序。
      6.  在 C 程序中，分号是语句结束符。也就是说，每个语句必须以分号结束。它表明一个逻辑实体的结束。
      7.  我对于编程的理解是，编程就是一堆字母，有固定语法，固定格式。 编程可以用于软件的制造，例如，微博可以通过代码制造出来，王者荣耀之类的游戏也可以通过软件制造出来。 还有就是你现在看到的网页。他还可以进行数据的统计，以及进行盗号（非法行为），还有就是2G，3G，5G网都由代码来实现。 另外，移动，联通，电信的流量无线，入侵别人电脑，访问外国网站，自己制作博客都是可以的。
      8.  关于编程这一块，想要熟练必须天天打代码，才可以。在我们这一块，入行经验很重要，入行越久，年薪越高。 不过，我们这一行的黄金年龄也就是35岁，过了35岁，脑力就开始退化，比较适合不太技术顶尖的职位。过于技术顶尖的职位太过于费脑子。
